# Gyanix 🌟
### Gyaan jo samjhe, aap ki baat

A beautiful AI chat platform powered by Sarvam AI.

## Tech Stack
- ⚡ Vite + React 18
- 🎨 Tailwind CSS
- 🌗 Dark / Light Mode
- 📱 Mobile Responsive
- 💬 Chat History (localStorage)
- ✨ Markdown + Code Highlighting

## Getting Started

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Push to GitHub
2. Import repo on vercel.com
3. Add environment variables
4. Deploy!

## Environment Variables

```env
VITE_SARVAM_API_KEY=your_api_key_here
```
